# Stalker Survivor

A top-down bullet heaven with STALKER-inspired aesthetics and manual reload as the core skill mechanic. Built in Python/Pygame.

## Concept
You are a lone survivor in an irradiated wasteland. Survive 20 minutes against escalating waves of bandits and mutants. Positioning and reload timing are everything.
